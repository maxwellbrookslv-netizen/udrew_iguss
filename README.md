# UDREW IGUSS - 你画我猜 React 版

这是一个基于 React + TypeScript 构建的网页端“你画我猜”游戏。

## 🚀 功能特点

- **实时画板**: 支持多种颜色选择、画笔粗细调节、橡皮擦以及一键清空画布。
- **游戏逻辑**: 包含倒计时系统、随机题目生成以及自动检测玩家猜测结果。
- **现代化 UI**: 使用 Tailwind CSS 打造，界面美观且响应迅速。
- **实时反馈**: 猜中题目时会有高亮提示并增加玩家积分。

## 🛠️ 技术栈

- **前端框架**: [React](https://reactjs.org/)
- **开发语言**: [TypeScript](https://www.typescriptlang.org/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **样式处理**: [Tailwind CSS](https://tailwindcss.com/)
- **图标库**: [Lucide React](https://lucide.dev/)

## 📦 快速启动

### 1. 克隆项目
```bash
git clone https://github.com/maxwellbrookslv-netizen/udrew_iguss.git
cd udrew_iguss
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```
